Job Boards Website


A Django-based job portal using the MVT architecture. The platform allows users to:

  • Sign up, log in, and manage their profiles

  • Employers can post jobs with details such as:
    • - Title

    • - Location

    • - Salary

    • - Job type


  • Applicants can apply directly through the platform

  • Includes job filtering for better user experience

  • Has a blog section with comments and search functionality

  • Features a contact page with email support

Built using Django's built-in authentication, class-based models, and template system.

Technologies Used

  • Backend Framework: Django

  • Architecture: MVT (Model-View-Template)

  • Authentication: Django built-in authentication

  • Templating: Django template engine

  • Models: Class-based models

  • Database: SQLite3

  • Support: Email support for contact and communication
