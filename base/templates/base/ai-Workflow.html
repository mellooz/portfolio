AI Workflow Platform

  • Platform Purpose: Contributed to a visual AI development platform where users can build complete AI models by connecting modular nodes (each representing an AI model or data operation) without writing code.

  • Backend Architecture: Used Controller–Service–Repository architecture for a clean and scalable backend structure, with workflows running in the backend and returning results to users.

  • Repository Layer: Focused on the Repository layer, implementing saving, loading, updating, and deleting node data, with file-based storage and database integration.

  • Chatbot API: Built the chatbot API using Django Channels, enabling real-time interactions between users and the AI engine.

  • Web Platform: Worked on the web platform allowing users to access tutorials, share projects, and browse community projects.

  • Authentication & Authorization: Implemented authentication and authorization, including login, OTP-based password reset, and role-based access.

  • Real-Time Collaboration: Enabled real-time chat rooms where messages are stored in the database, and users can collaborate in project-specific rooms.
