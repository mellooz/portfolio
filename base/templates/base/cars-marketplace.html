Car Marketplace

  • Project Type: Django-based web application

  • Main Features: Manage user profiles, car listings, and locations

  • Functionality:
    • User registration and profile management

    • Create, post, filter, and like car listings


  • Implementation:
    • Used models, forms, and signals

    • Integrated custom widgets

    • Optimized data flow with Django signals


  • Goal: Seamless and interactive user experience in the marketplace

Technologies Used

  • Framework: Django

  • Architecture: MVT (Model-View-Template)

  • Authentication: Django built-in authentication system

  • Features: Django models, forms, signals, and custom widgets

  • Interactivity: Django signals for real-time behavior and data flow

  • Frontend: HTML, CSS and Bootstrap

  • Database: SQLite3
