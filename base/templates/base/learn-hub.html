LearnHub


LearnHub API is a backend service built with Django and designed to support an e-learning platform with JWT and OTP-based authentication.

Users can:

  • Enroll in courses
  • Search for content
  • Create and manage their own courses with lessons
  • Track their activity via a personalized profile

The system ensures secure account verification and password reset through email-based OTPs.

Technologies Used

  • Backend Framework: Django
  • API Format: REST API
  • Authentication: JWT, OTP (Email-based)
  • Database: Sqlite3
  • Email Services: SMTP for OTP verification and password reset
